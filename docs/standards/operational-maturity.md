# Operational Maturity

This standard defines continuous operational improvement.

## Incident Drills

- Quarterly incident response drills
- Post-drill review with action items

## SLO Reviews

- Quarterly SLO review and adjustments
- Track error budget burn rates
- Record review outputs and adjustments in the [Service Ownership Checklist](../templates/service-ownership-checklist.md) so the next owner sees what changed.

## Post-Mortem Trends

- Review incident trends quarterly
- Track repeat failures and remediation
- Feed remediation items into the release readiness checklist and runbook updates for recurring issues.
