# Supply Chain and SBOM Standard

This standard defines software supply chain security requirements.

## SBOM

- Generate SBOM for every build (CycloneDX)
- Store SBOM artifacts with release metadata

## Provenance

- Signed container images required
- Build provenance recorded in CI

## Vulnerability SLAs

- Critical: fix within 7 days
- High: fix within 30 days
- Medium: fix within 90 days
