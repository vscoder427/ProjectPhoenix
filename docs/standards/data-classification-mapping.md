# Data Classification Mapping

This standard defines what data is classified as PHI/PII for Employa.

## PHI Examples

- Medical history
- Treatment status
- Recovery program participation

## PII Examples

- Full name
- Email address
- Phone number
- Home address

## Internal

- Employer notes
- Non-public business metrics

> Align mappings with the [Data Classification and Retention](data-classification-retention.md) standard and update the inventory when new types are introduced.
